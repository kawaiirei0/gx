## 🚀 快速链接

### 新用户

如果你是第一次使用 gx：

1. 阅读 [QUICKSTART.md](../QUICKSTART.md) 快速上手（推荐）
2. 或阅读 [README.md](../README.md) 了解项目概览
3. 按照 [INSTALLATION.md](../INSTALLATION.md) 安装 gx
4. 参考 [COMMANDS.md](../COMMANDS.md) 了解详细命令

### 贡献者

如果你想为 gx 做贡献：

1. 阅读 [CONTRIBUTING.md](../CONTRIBUTING.md) 了解贡献流程
2. 查看 [ARCHITECTURE.md](../ARCHITECTURE.md) 理解项目架构
3. 运行 [examples/](../examples/) 中的示例程序
4. 开始你的第一个 Pull Request

### 开发者

如果你想使用 gx 的内部 API：

1. 查看 [ARCHITECTURE.md](../ARCHITECTURE.md) 了解组件设计
2. 阅读 [examples/README.md](../examples/README.md) 学习 API 使用
3. 参考示例程序编写代码
4. 查看接口定义 (`pkg/interfaces/`)
